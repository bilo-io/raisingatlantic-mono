# **App Name**: Raising Atlantic

## Core Features:

- User Authentication: Secure sign-up, login, and role-based access control (RBAC) for Parents, Clinicians, and Admins.
- Child Profile Management: Parents can create, view, and manage child profiles, including notes and avatar uploads.
- Landing Page: Marketing page with sections for hero, services, features, FAQs, testimonials, and lead capture.
- Multi-language Support: Implemented using react-i18next for easy translation and localization.
- Responsive Navigation: Sidebar navigation with role-based links, adapting to mobile with a hamburger menu.

## Style Guidelines:

- Primary color: Soft blue (#605BFF) to evoke a sense of calm and reliability, suitable for a platform supporting child development.
- Background color: Very light blue (#E8F4F8), subtly related to the primary hue, to create a soothing backdrop that doesn't distract from content.
- Accent color: Muted green (#FF00AA) to highlight interactive elements and CTAs, promoting growth and positive action.
- Body text and headlines: 'PT Sans', sans-serif for clear readability and a modern feel.
- Simple, friendly icons representing various aspects of child development and user roles.
- Clean and intuitive layout with a sidebar for main navigation and clear sections for child profile management.
- Subtle transitions and animations to enhance user experience and provide feedback on interactions.
